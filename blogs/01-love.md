# How to Use README.md Blogs in Your Portfolio

Using **README.md (Markdown)** blogs is a simple way to showcase writing and code in your portfolio. Markdown is clean, readable, and easy to manage via GitHub.

---

## 1. Create a GitHub Repository or Folder

1. Go to [GitHub](https://github.com/) and log in.  
2. Create a repo or use your portfolio repo (e.g., `my-portfolio-blogs`).  
3. Add a folder called `blogs` to store Markdown files.

---

## 2. Write Your Blogs in Markdown

- Markdown supports **headings, lists, code blocks, images, links**.  
- Example:

```javascript
console.log("Hello, Markdown Blog!");
```
